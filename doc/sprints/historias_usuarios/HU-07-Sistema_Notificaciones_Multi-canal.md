## HU-07: Sistema de Notificaciones Multi-canal

**Como** donante recurrente  
**Quiero** recibir notificaciones sobre mi donación  
**Para** tener transparencia en el proceso. Las personas pueden mandar mensajes igual a cualquier apartado de mensajería de otras páginas.

### Criterios de Aceptación

1. El sistema debe enviar notificaciones cuando:
   - Se registra una nueva donación
   - Los productos son distribuidos
   - Se completa el ciclo de una donación

2. Los usuarios deben poder elegir el canal de notificación preferido:
   - Email
   - SMS
   - Notificación en la aplicación web

3. El sistema de mensajería debe permitir:
   - Enviar mensajes entre usuarios registrados
   - Ver historial de conversaciones
   - Recibir notificaciones de mensajes nuevos
   - Adjuntar archivos (máximo 5MB)

4. Las notificaciones deben incluir información relevante y específica de cada evento.

5. Los usuarios deben poder silenciar o personalizar sus preferencias de notificación.

6. El sistema debe garantizar la entrega de notificaciones críticas independientemente de las preferencias.
