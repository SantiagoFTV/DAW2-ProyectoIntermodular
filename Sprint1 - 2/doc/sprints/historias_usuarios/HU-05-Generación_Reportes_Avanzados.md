## HU-05: Generación de Reportes Avanzados

**Como** director de la organización  
**Quiero** generar reportes estadísticos detallados  
**Para** la toma de decisiones estratégicas. Se podrá filtrar las fechas para ver diferentes periodos, ver el porcentaje de productos mayormente donados y se podrá descargar un excel con esta información.

### Criterios de Aceptación

1. El sistema debe ofrecer filtros de reporte por:
   - Rango de fechas personalizables
   - Tipo de producto
   - Punto de distribución
   - Donante

2. El reporte debe incluir:
   - Total de donaciones recibidas
   - Total de productos distribuidos
   - Productos en inventario
   - Porcentaje de productos por categoría
   - Número de beneficiarios atendidos

3. Los reportes deben incluir gráficos visuales:
   - Gráfico de barras o circular para productos más donados
   - Gráfico de líneas para evolución temporal

4. Debe ser posible exportar el reporte en formato Excel (.xlsx).

5. El archivo Excel debe incluir múltiples hojas con datos detallados y resúmenes.

6. Los reportes deben generarse en menos de 10 segundos para rangos de hasta 1 año.
