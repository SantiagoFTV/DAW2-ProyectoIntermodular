## HU-09: Gestión de Voluntarios

**Como** coordinador de voluntarios  
**Quiero** gestionar el registro y asignación de voluntarios  
**Para** optimizar los recursos humanos. Quiénes son las personas registradas y sus horarios para poder ayudar con sus habilidades. Este apartado tiene botones para agregar, quitar o modificar a los voluntarios.

### Criterios de Aceptación

1. El sistema debe permitir registrar voluntarios con:
   - Datos personales (nombre, apellidos, DNI, contacto)
   - Habilidades y experiencia
   - Disponibilidad horaria
   - Días disponibles

2. Debe existir una vista de lista con todos los voluntarios registrados mostrando:
   - Nombre completo
   - Estado (Activo/Inactivo)
   - Próxima disponibilidad
   - Habilidades principales

3. El sistema debe incluir botones funcionales para:
   - Agregar nuevo voluntario
   - Editar información de voluntario existente
   - Eliminar/desactivar voluntario
   - Ver historial de participación

4. Debe ser posible buscar y filtrar voluntarios por:
   - Nombre
   - Habilidades
   - Disponibilidad
   - Estado

5. El sistema debe validar que no haya duplicados al registrar un nuevo voluntario (por DNI).

6. Al eliminar un voluntario, debe solicitarse confirmación para evitar borrados accidentales.
